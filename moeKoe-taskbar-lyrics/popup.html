使用教程

1. 开启 MoeKoeMusic API 模式

打开 MoeKoeMusic 设置 → 系统 → API模式 → 开启

2. 首次启动

打开 MoeKoeMusic 设置 → 插件 → 打开插件目录 → 打开 moeKoe-taskbar-lyrics 文件夹 → 双击 MoeKoeTaskbarLyrics.exe。首次启动完成后，建议打开“开机自动启动”，若已开启则关闭此选项后再次打开此选项

3. 调整位置

鼠标悬停歌词窗口 → 按住左键拖动调整位置
