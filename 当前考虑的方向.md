现在最建议做的事情

先把设置系统做好。

因为用户最先吐槽的一定是：

歌词很好

但是：
字体太小
颜色不好看
位置不对
显示太长
重启丢设置

而不是：

为什么不是嵌入任务栏
第一优先级
配置持久化

建议做：

{
  "enabled": true,
  "fontFamily": "Microsoft YaHei UI",
  "fontSize": 16,
  "fontWeight": 500,

  "normalColor": "#FFFFFF",
  "highlightColor": "#4CC2FF",

  "opacity": 1.0,

  "offsetX": 0,
  "offsetY": 0,

  "showNextLine": true,

  "autoHide": false
}

存放：

%APPDATA%\MoeKoeTaskbarLyrics\config.json
第二优先级
字体设置

必须有：

字体
字号
粗细

例如：

微软雅黑
Maple Mono
HarmonyOS Sans

很多用户会非常在意。

第三优先级
颜色设置

至少：

普通歌词颜色
高亮歌词颜色

例如：

白色
天蓝
网易云红
QQ音乐绿

预设主题。

第四优先级
位置调整

很多人任务栏情况不同：

开始 搜索 ...
↑
歌词太靠左

所以需要：

水平偏移
垂直偏移

最好支持：

拖动歌词直接调整

保存到配置。

第五优先级
实时预览

不要：

改配置
↓
确定
↓
重启

而是：

拖动字号
↓
歌词立即变化

体验会好很多。

设置窗口建议

不要自己用 Win32 拼。

你现在已经用了：

Win32
Direct2D

我会直接用：

WebView2

作为设置界面。

结构：

主程序
│
├─ Lyrics Window
├─ Tray Icon
└─ Settings Window(WebView2)

设置页：

主题
字体
字号
颜色
透明度

[应用]

然后：

ICoreWebView2

和 C++ 通信。

优点：

UI瞬间现代化
开发速度快
后续加配置很容易
支持暗色模式

很多现代 Windows 工具都这么干。
