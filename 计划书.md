## 本项目是一个 MoeKoeMusic Windows 客户端的插件，功能是任务栏歌词。

## 1. 实现“插件”思路：

&#x20;

**外挂方式**

- 不改动原程序本体，通过读取播放器状态、歌词数据来显示任务栏歌词。
- 优点：不破坏原程序，可随更新独立维护。
- 缺点：需要解析程序内存或接口来获取播放信息。

## 2. Windows 任务栏歌词显示实现

&#x20;

任务栏歌词表现方式：

**任务栏悬浮窗（Overlay）**

- 在任务栏上方绘制透明窗口，显示实时歌词。
- 技术点：
  - 使用 `Win32 API` 创建 **无边框透明窗口**。
  - 设置 `WS_EX_TOPMOST | WS_EX_LAYERED | WS_EX_TRANSPARENT` 让窗口透明且点击穿透。
  - 结合 `Direct2D` 或 `GDI+` 绘制文字，保证渲染流畅。
- 优点：效果酷，可实时滚动显示。
- 缺点：稍复杂，需要处理高DPI、多屏幕、任务栏位置（底/左/右/上）。

## 3. 获取歌词数据

&#x20;

MoeKoeMusic 的 Windows 版本是 Flutter + Rust/Go 后端：

- **Flutter 前端**可能通过网络接口获取歌词，可以：
  1. **读取歌词缓存文件**（如果存在）。
  2. **Hook WebSocket/HTTP API**，获取当前播放歌曲及时间戳。
  3. **内存 Hook**（高级）获取当前播放进度和歌曲信息。
- 选择方案：
  - 推荐先从 **歌词缓存文件或API** 入手，简单易维护。
  - 避免直接内存 Hook，兼容性差，容易崩溃。

## 4. 技术栈建议

| 层     | 技术                      |
| :---- | :---------------------- |
| 外挂主程序 | C++ （WinForms）          |
| 窗口绘制  | Win32 + GDI+ / Direct2D |
| 歌词处理  | UTF-8 字符串处理，支持 LRC 时间同步 |
| 定时刷新  | 高精度 Timer 或消息循环         |

<br />

### 使用 **C++ + Win32** 可以更轻量、CPU 占用更低，但绘制和动画要自己管。

## 5. 用户体验建议

&#x20;

- 可选透明度和字体颜色，让任务栏歌词不遮挡图标。
- 支持歌词同步滚动或高亮当前行。
- 支持 **任务栏隐藏时自动隐藏歌词**。
- 可以增加 **鼠标悬停显示完整歌词**。

## 第二阶段（正式版）

&#x20;

增加：

### 歌词动画

```
上一句淡出
      ↓
当前句高亮
      ↓
下一句淡入
```

类似：

```
♪ 夜空中最亮的星
   ↑ 当前句

  给我再去相信的勇气
```

### 卡拉OK效果

&#x20;

类似网易云桌面歌词：

```
[████████░░░░░░░]
夜空中最亮的星
```

根据播放进度逐字高亮。

### 自动吸附任务栏

&#x20;

支持：

```
底部任务栏
顶部任务栏
左侧任务栏
右侧任务栏
```

获取：

```
SHAppBarMessage()
```

即可获得任务栏位置。
