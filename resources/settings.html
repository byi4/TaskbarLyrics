使用教程

1. 开启 MoeKoeMusic API 模式

打开 MoeKoeMusic 设置 → 系统 → API模式 → 开启

2. 安装插件

打开 MoeKoeMusic 设置 → 插件 → 安装插件 → 选择解压后的本项目 release 文件夹

3. 首次启动

打开 MoeKoeMusic 设置 → 插件 → 打开插件目录 → 打开 moeKoe-taskbar-lyrics 文件夹 → 双击 MoeKoeTaskbarLyrics.exe

4. 调整位置

鼠标悬停歌词窗口 → 按住左键拖动调整位置
